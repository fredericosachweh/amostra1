
1. OSD to set resolution
caResChanger.toSetResolution(  "800x600" );

PS. A. the resolution value is not restricted and free to set to any value at this moment.

2. to restart arora

caResChanger.toSetParameter( "xxx.html" );
caResChanger.toQuit();

PS. A.The "toSetParameter()" function must be set before calling "toQuit", otherwise you will get a blank screen.
B. xxx.html is the page you want to show on next start. Ex. It can be index.html or a settings html.

1. to change volume,
        caMediaPlayer.toSetVolume( value );

        PS. The default value is 29 , and the max value is 33

        2. to set volume to mute directly
        caMediaPlayer.toSetVolumeMute();
